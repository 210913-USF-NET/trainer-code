# Bash
Bourne-again shell

## commands
- cd: Change directory
- ls: list contents
- pwd: present working directory (where you're right now)
- mkdir: Make new directory
- touch: makes new files
- rm: deletes stuff
    - ```rm -r``` for deleting folders
- **mv**: moves files/directories
    - ```mv currPath newPath```
    - also use mv to rename
- **cp**: Copies stuff
    - ```cp currfile newFileName```
